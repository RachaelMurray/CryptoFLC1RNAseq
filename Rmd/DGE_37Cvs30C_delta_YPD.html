Summary

Differential Gene Expression 37°C vs 30°C in delta in YPD samples, using DESeq2

This analysis produces files:

  • deseq_df_37Cvs30C_delta_YPD.txt
  • deglist_37Cvs30C_delta_YPD_up2x_FDR5.txt
  • deglist_37Cvs30C_delta_YPD_down2x_FDR5.txt

Load count data from shared script load_count_data.Rmd

Load and organise count data

load_count_data.Rmd is just to load and organise the raw count data in R, as well as loading libraries for data analysis. Other scripts can just run this one instead of duplicating code, especially scripts for differential gene expression analysis.

This runs after QC_PCA.Rmd, which does the organisation. This just loads the organised counts from the file results/summaries/counts_bysamplecode.txt.

  biobroom::tidy.DESeqDataSet(dds_37Cvs30C_delta_YPD) %>%
  dplyr::filter(term == "Temperature_37_vs_30") %>%
  dplyr::left_join(gene_info_df,

  dplyr::select(gene, 
                geneName, 
                baseMean, 
                log2FC = estimate, 
                stderror,

Volcano plot of DEGs

volcano_37Cvs30C_delta_YPD <- 
  ggplot(data = deseq_df_37Cvs30C_delta_YPD,
         aes(x = log2FC, y = -log10(padj)) ) +
  geom_point(size = 0.5, colour = "grey50") +
  geom_point(data = DEGdf_WT_up2x_FDR5,
             size = 1, colour = "darkblue") +
  geom_point(data = DEGdf_WT_down2x_FDR5,
             size = 1, colour = "darkred") +
  scale_y_continuous("-log10(p)",
                     limits = c(0,20), expand = c(0,0),
                     oob=scales::squish) +
  scale_x_continuous("log2 fold-change, 37°C vs 30°C, FLC1∆, YPD",
                     breaks = -5:5,
                     limits = c(-4,4), expand = c(0.01,0.01),
                     oob = scales::squish) +
  coord_cartesian(clip = "off")
